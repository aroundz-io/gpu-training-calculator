# TrainLab — 온라인 AI 학습 서비스 (데모)

학습 데이터를 업로드하고 결제하면 바로 학습이 시작되는 구조의 온라인 AI 학습 서비스 데모입니다.
정적 파일(HTML/CSS/JS)만으로 구성되어 있으며, 실서비스 전환 시 업로드·결제·작업 큐만 백엔드로 교체하면 됩니다.

**Live**: https://gpu-training-calculator.vercel.app

## 서비스 플로우

1. **데이터 업로드** — 드래그&드롭으로 파일 선택. 텍스트/이미지 자동 판별, 토큰 수·용량 환산 (데모: 파일은 서버 전송 없이 브라우저에서만 분석)
2. **모델 · 학습 설정** — 오픈소스 모델 프리셋, 학습 유형(Full FT/LoRA/사전학습/비전/확산), 에포크, 정밀도
3. **GPU 선택 · 견적** — GPU 28종 중 선택, 💰최저 비용/⚡최단 시간 추천 배지, 원화 견적(부가세 포함), 메모리 부족 경고
4. **결제 → 학습 시작** — 모의 카드 결제 후 작업 등록
5. **대시보드** — 진행률, loss 곡선, 학습 로그, GPU 사용률, 누적 비용 실시간 표시 (실제 예상 시간을 40~150초로 압축한 시뮬레이션, localStorage 저장), 완료 시 체크포인트 메타 다운로드

## 회원 · 관리자

- **회원가입/로그인** (데모: localStorage) — 결제는 회원만 가능하며, 비회원은 결제 단계에서 가입 안내 게이트가 표시됩니다.
- **처리 우선순위** — 표준 / 높음(+15%) / 긴급(+30%) 큐 배정, 요금 할증이 견적에 반영됩니다.
- **관리자 콘솔** — 메뉴에 노출되지 않고 `/#admin` 주소로만 접근 (데모 계정: `admin` / `trainlab2026`).
  통합 대시보드: 회원 수·학습량·매출(일/주/월)·GPU 클러스터 가동률·작업 큐 우선순위 관리·서버 노드 관리·스토리지 사용량/보관 목록·최근 결제 내역.
- 기준 환율: **1,540원/달러** (계산기에서 조정 가능)

## 데이터 보관 정책

- 업로드한 학습 데이터는 학습 수행을 위해 **임시 보관**되며, 학습이 끝난 후에는 삭제 조치됩니다.
- 스토리지 서비스 미이용 시 **학습 완료 15일 후 영구 삭제**를 원칙으로 하며, 삭제된 데이터는 복구할 수 없습니다.
- 데이터를 계속 보관하려면 **별도의 유료 스토리지 서비스**(GB당 월 40원, 최소 월 500원)를 이용해야 하며, 이용 기간 동안 안전하게 보관됩니다.
- 위 정책은 홈 안내, 업로드 단계 보관 옵션, 결제 약관 동의, 대시보드 보관 상태(삭제 예정일 D-day), 이용약관 전문(제4조)에 명시되어 있습니다.

## 비용 계산기 기능

- **학습 유형 6종**: LLM 사전학습 / 전체 파인튜닝 / LoRA·QLoRA, 이미지 분류(CNN·ViT), 확산 모델(SD 계열), 기상·지구 시뮬레이션(NVIDIA Earth-2: FourCastNet 3, CorrDiff)
- **유명 오픈소스 모델 프리셋**: NVIDIA Nemotron, Llama 3.1, DeepSeek-V3/R1, Qwen, Mixtral, Gemma, Phi-4, gpt-oss 등 (MoE는 활성 파라미터 기준 연산)
- **NVIDIA GPU 28종 비교**: GB300/GB200/B200/B100, H200/H100/H20, L-시리즈, A100 등 Ampere·Volta·Turing, DGX Spark, RTX PRO/워크스테이션, GeForce RTX 30/40/50
- 데이터 양을 토큰 수와 **텍스트 용량(GB/TB)** 으로 상호 환산 표시
- BF16/FP8 정밀도, MFU(실효 효율), GPU 개수, 원/달러 환율 조정
- GPU별 시간당 단가 직접 수정 가능, 메모리 부족 경고

## 계산 방식

- LLM: `총 FLOPs ≈ 6 × 파라미터 × 토큰` (LoRA ≈ 0.67배, MoE는 활성 파라미터 기준)
- 비전/확산/기상: `샘플당 순전파 FLOPs × 3 × 샘플 수 × 에포크`
- 학습 시간 = `총 FLOPs ÷ (GPU 수 × 이론 TFLOPS × MFU)`
- Earth-2 프리셋은 공개된 학습 비용(FCN3 ≈ 9만 GPU·시간, CorrDiff ≈ 5,000 A100 GPU·시간)에 맞춰 보정

> ⚠️ 모든 수치는 추정치입니다. 실제 학습 시간은 데이터 파이프라인, 통신 병목, 시퀀스 길이 등에 따라 달라질 수 있으며, 시간당 단가는 2025~2026년 온디맨드 시세 기준 기본값입니다.

## 실행

```bash
python -m http.server 4173
```
